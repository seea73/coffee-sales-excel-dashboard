## üìä Dashboard Features

- **Key Performance Indicators (KPIs)**:
  - Total Sales
  - Total Profit
  - Total Quantity Sold

- **Visual Charts**:
  - Sales Trend Over Time (Line Chart)
  - Sales by Country (Bar Chart)
  - Profit by Coffee Type and Roast
  - Top 5 Customers by Sales

- **Interactivity**:
  - Slicers for:
    - Date
    - Coffee Type
    - Roast
    - Size
    - Loyalty Card Status
